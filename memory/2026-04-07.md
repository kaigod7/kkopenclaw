## 今日记录

- K 装了 claude-code（@anthropic-ai/claude-code v2.1.92）
- 安装过程中遇到地区限制问题：Anthropic API 在中国内地不提供服务
- flclash 规则模式 terminal 流量不转发代理，导致 claude 连不上
- 解决：~/.zshrc 添加两行环境变量，让 terminal 流量走 127.0.0.1:7890
  ```bash
  export http_proxy=http://127.0.0.1:7890
  export https_proxy=http://127.0.0.1:7890
  ```
